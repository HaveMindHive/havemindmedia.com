## Game Mechanics

### Triad Matching
- Touch 3 adjacent symbols of compatible types
- NOT like Candy Crush doubling - just the exact 3 you select
- If multiple triads possible from your selection = bonus points

### Balance Score
Must maintain ratio:
- S- (love/soul): MOST
- Coin (observer): MIDDLE
- S+ (physical): LEAST (but not zero)

Example ideal: 5 S- : 3 Coin : 2 S+ (ratio 5:3:2)

If unbalanced = game slows, harder to see, eventually lose

### Push Mechanic
- Rows push toward you from top
- Like quarter-pusher arcade game
- Clear triads before they reach bottom
- Each level adds more starting rows

## Level Structure

### STAGE 1: Learning (Levels 1-10)
- Start with 16 rows
- Add 1 row per level (16â†’17â†’18...â†’25 rows at level 10)

- "Did You Know?" geometry facts between levels

### STAGE 2: Inversion (Levels 1-10)
- Same symbols but INVERTED meaning
- What looked like S+ now means S-
- Must relearn the visual language
- Rows: 16 + (2,3,4,6,7,8,9,8) per level
- Level 10 has ONE unsolvable Q
- Q appears impossible until they're frustrated
- Then Q expands, surrounded by q variants
- Remove the Q to advance
- Evil laugh when Q is revealed
- Only 3 total restarts for entire stage

### STAGE 3: VOID (Final Boss)
- ONE SHOT - fail means total restart
- 22 layers pushing
- 8-second cycles:
  1. Normal (2 sec calm, easy to see)
  2. Chromatic inversion (8 sec)
  3. Spatial inversion + upside down (8 sec)
  4. 2 seconds of CHAOS (all switching, inverting, recursive)
  5. Stops on whatever state
- Q and q characters present
- Must maintain balance while reality shifts
