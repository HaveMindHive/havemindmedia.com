The Proof

Let's prove that four unit vectors at 0°, 90°, 180°, 270° always sum to zero.

Theorem: Quadrature Balance

1
v₁ = (cos 0°, sin 0°) = (1, 0)
First vector points right
2
v₂ = (cos 90°, sin 90°) = (0, 1)
Second vector points up
3
v₃ = (cos 180°, sin 180°) = (-1, 0)
Third vector points left
4
v₄ = (cos 270°, sin 270°) = (0, -1)
Fourth vector points down
5
v₁ + v₂ + v₃ + v₄ = (1+0-1+0, 0+1+0-1) = (0, 0)
Sum the x-components: 1 + 0 + (-1) + 0 = 0
Sum the y-components: 0 + 1 + 0 + (-1) = 0
Q.E.D.

Four vectors at 90° intervals sum to exactly zero.

The General Principle

Quadrature is just one case of a more general rule:

Σ ei·2πk/n = 0 for k = 0, 1, ..., n-1
n evenly-spaced unit vectors sum to zero (n ≥ 2)
  • n = 2: Two opposite vectors (0°, 180°) → sum = 0
  • n = 3: Three vectors at 120° → sum = 0 (3-phase power)
  • n = 4: Four vectors at 90° → sum = 0 (quadrature)
  • n = 6: Six vectors at 60° → sum = 0 (6-cylinder engine)

This is why symmetric arrangements always balance. It's not coincidence — it's geometry.

Symmetry creates balance.
The circle is the source of both.
