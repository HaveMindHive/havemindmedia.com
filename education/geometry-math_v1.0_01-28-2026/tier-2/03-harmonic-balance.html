The Problem: Vibration

Imagine an engine with pistons going up and down. Each piston pushes in one direction, creating a force. If all pistons pushed at the same time, the engine would shake violently.

The solution? Spread the pushes around the circle.

Watch the four pistons. They're at 0°, 90°, 180°, 270° — evenly spaced.

Phase Quadrature: The Magic Arrangement

When you place four things at 0°, 90°, 180°, and 270° around a circle, something special happens:

0°
sin(0°) = 0
90°
sin(90°) = 1
180°
sin(180°) = 0
270°
sin(270°) = -1

Add them up:

0 + 1 + 0 + -1 = 0

Four phases at 90° apart always sum to zero.
The forces cancel. The vibrations cancel. Perfect balance.

The Pattern

Any number of evenly-spaced points around a circle will balance:

  • 2 points at 180° apart: sin(0°) + sin(180°) = 0 + 0 = 0 ✓
  • 3 points at 120° apart: sin(0°) + sin(120°) + sin(240°) = 0 ✓
  • 4 points at 90° apart: sin(0°) + sin(90°) + sin(180°) + sin(270°) = 0 ✓
  • n points at 360°/n apart: Always sums to 0 ✓

This is a mathematical certainty. The geometry guarantees it.
