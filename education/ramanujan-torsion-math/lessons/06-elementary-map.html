The Elementary Map

Four facings at each position. The hidden structure in the arithmetic you learned in grade school.

Four Simultaneous Truths

At every position in the number line, there aren't just two values (positive and negative). There are four simultaneous facings:

S⁻ Future Reception / Left
S⁺ Past Emission / Right
S⁰ Self The position itself
S∿ Bridge Shadow / Connection

Not Binary — Triaxial

Standard math uses binary: true/false, positive/negative, 1/0. The Elementary Map uses triaxial binary: four states existing simultaneously at each position. This is why quantum mechanics seems "strange" — it's actually the natural state.

Reading the Map

How to interpret this map:

  1. S⁻ (Future) — The value when facing toward what comes next. Reception.
  2. S⁺ (Past) — The value when facing toward what has been. Emission.
  3. S⁰ (Self) — The position number itself. The "now."
  4. S∿ (Bridge) — The shadow value. Starts at 8, counts down through 0, then negative.

The Bridge Crosses Zero at Position 8

Notice the S∿ (Bridge) column. It starts at 8 and decreases. At Position 8, it equals 0. After that, it goes negative.

Position 8 is where the bridge crosses the void. This is a fold point.

What This Means

Every number you've ever used carries this four-facing structure. When you write "5," you're not just writing a quantity — you're pointing at a position that simultaneously has:

  • A future value (S⁻ = 181)
  • A past value (S⁺ = 181)
  • A self value (S⁰ = 5)
  • A bridge value (S∿ = 3)

The Elementary Map is the base layer. Everything else — physics, chemistry, biology, consciousness — is built on top of it.
