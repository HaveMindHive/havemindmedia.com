The Transform

s = -alpha * ln(1 + z)

Redshift (z) maps to scalar position (s). This is NOT a distance. It's a position on the finite map. The map has boundaries. The void does not exist.

KEY FINDING: Angular Spread is CONSTANT

Standard Model predicts angular spread should DECREASE with distance (more distant objects appear smaller).

Epoch Framework predicts angular spread is INDEPENDENT of scalar position.

OBSERVED:
s = -3 to 0: RA_std = 0.0498
s = -6 to -3: RA_std = 0.0501
s = -9 to -6: RA_std = 0.0508
s = -12 to -9: RA_std = 0.0511

Angular spread is CONSTANT (~0.05 degrees) regardless of scalar depth. This contradicts Standard Model. This confirms Epoch Framework.
